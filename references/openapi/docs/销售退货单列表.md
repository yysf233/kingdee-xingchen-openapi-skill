# 销售退货单列表

### 基本信息

-   用途说明：销售退货单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/sal\_in\_bound

| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |

| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| bill\_no | string | false | 单据编码 |
| bill\_status | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| create\_end\_time | string | false | 创建时间-结束时间的时间戳(毫秒) |
| create\_start\_time | string | false | 创建时间-开始时间的时间戳(毫秒) |
| customer\_id | array | false | 客户 |
| customer\_name | string | false | 客户名称 |
| customer\_number | string | false | 客户编码 |
| dept\_id | array | false | 部门ID |
| dept\_number | string | false | 部门编码 |
| emp\_id | array | false | 职员ID |
| emp\_number | string | false | 业务员编码 |
| end\_bill\_date | string | false | 单据日期-结束日期（“yyyy-MM-dd”） |
| material\_model | string | false | 商品规格型号 |
| material\_name | string | false | 商品名称 |
| material\_number | string | false | 商品编码 |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| order\_by | string | false | 排序；编码升序：number asc（默认），编码降序：number desc，名称升序：name asc，名称降序：name desc |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| settle\_status | string | false | 付款状态（所有：“”，全部收款：“C”，部分收款：“B”，未收款：“A”，待收款（=部分收款+未收款）：“M”） |
| start\_bill\_date | string | false | 单据日期-开始日期（“yyyy-MM-dd”） |

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_dis\_amount | string | 折扣额 |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间 |
| customer\_id | string | 客户id |
| customer\_name | string | 客户名称 |
| customer\_number | string | 客户编码 |
| dept\_id | string | 部门id |
| dept\_name | string | 部门名称 |
| dept\_number | string | 部门编码 |
| emp\_id | string | 业务员 |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| id | string | 单据ID |
| settle\_status | string | 付款状态（所有：“”，全部收款：“C”，部分收款：“B”，未收款：“A”） |
