# 委托代销列表

### 基本信息

-   用途说明：委托代销列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/entrust

| delivery\_type | string | false | 交货方式编码（所有：“”，物流发货：“JHFS01”，车辆配送：“JHFS03”，客户自提：“JHFS04”） |
| delivery\_type\_id | string | false | 交货方式ID（所有：“”，物流发货：“6”，车辆配送：“8”，客户自提：“9”） |
| deliveryman\_ids | array | false | 配送员ID |

| dept\_number | string | false | 部门编码 |

| end\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），结束日期 |
| end\_time | string | false | 结束时间（格式：“时间戳”，-1表示不过滤），创建时间 |
| expire\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| expire\_time | string | false | 结束时间（格式：“时间戳”，-1表示不过滤），修改时间 |
| express\_company\_ids | array | false | 物流方式 |
| filter\_zero | string | false | 过滤字段值为0记录：目前仅支持totalunsettleamount |

| material\_name | string | false | 商品名称 |
| material\_number | string | false | 商品编码 |

| start\_time | string | false | 开始时间（格式：“时间戳”，-1表示不过滤），创建时间 |
| stock\_id | array | false | 仓库 |
| total\_fields | string | false | 合计值字段，多个字段用英文逗号隔开，支持的字段：total（默认：价税合计）、totalunsettleamount（未收金额）、totalinsamount（收款合计）、totalamount（本次应收账款） |

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_dis\_amount | string | 整单折扣额 |
| bill\_dis\_before\_amount | string | 单据整单折前价税合计 |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间 |
| customer\_id | string | 客户id |
| customer\_name | string | 客户名称 |
| customer\_number | string | 客户编码 |
| delivery\_status | string | 配送状态（所有：“”，全部配送：“A”，部分配送：“P”，未配送：“N”,无需配送：“C”） |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| id | string | 单据ID |
| settle\_status | string | 付款状态（所有：“”，全部收款：“C”，部分收款：“B”，未收款：“A”） |
| total\_amount | string | 单据价税合计 |
