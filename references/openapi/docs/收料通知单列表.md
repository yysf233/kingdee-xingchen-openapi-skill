# 收料通知单列表

### 基本信息

-   用途说明：收料通知单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/pur\_mat\_rec\_notice

### 请求参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |

| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| barcode | string | false | 商品条码 |
| bill\_close\_state | string | false | 单据关闭状态（所有：“”，未关闭：“C”，已关闭：“S”,手动关闭: “H”） |
| bill\_status | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| bill\_stock\_id | array | false | 表头仓库ID |
| create\_end\_time | string | false | 创建时间-结束时间的时间戳(毫秒) |
| create\_start\_time | string | false | 创建时间-开始时间的时间戳(毫秒) |
| creator\_id | array | false | 制单人，格式：\[“制单人id”,“制单人id”\] |
| dept\_id | array | false | 部门ID |
| emp\_id | array | false | 职员ID |
| io\_status | string | false | 入库执行行状态（所有：“”，全部执行：“C”，部分执行：“Z”，未执行：“A”） |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| sp\_id | array | false | 表体仓位ID |
| stock\_id | array | false | 表体仓库ID |

--header 'X-Api-Nonce: 1655775240000' \

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| headers | Header | 返回数据头信息 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_close\_state | string | 单据关闭状态（未关闭：“C”，已关闭：“S”，手动关闭：“H”） |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_dis\_amount | string | 整单折扣额 |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间（格式 yyyy-MM-dd HH:mm:ss） |
| custom\_field | Map <string> | 自定义字段[使用指南](https://open.jdy.com/#/files/api/detail?id=76567ff2a06311edaa4b3d71bf0fce53&noside=true) |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| id | string | 单据ID |
| io\_status | string | 入库执行状态（全部执行：“C”，部分执行：“Z”，未执行：“A”） |
| material\_entity | List <MaterialEntity> | 商品分录 |
| modify\_time | string | 修改时间（格式 yyyy-MM-dd HH:mm:ss） |
| supplier\_id | string | 供应商id |
| supplier\_name | string | 供应商名称 |
| supplier\_number | string | 供应商编码 |
| total\_amount | number | 单据价税合计 |

**MaterialEntity**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| custom\_entity\_field | Map <string> | 自定义字段[使用指南](https://open.jdy.com/#/files/api/detail?id=76567ff2a06311edaa4b3d71bf0fce53&noside=true) |
| id | string | 分录ID |

**Header**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| all\_amount | string | 价税合计 |
| total\_amount | string | 列表价税合计汇总，符合条件所有单据价税合计汇总，包含本页之外的数据 |
