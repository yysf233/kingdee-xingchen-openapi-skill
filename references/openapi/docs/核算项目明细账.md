# 核算项目明细账

### 基本信息

-   用途说明：核算项目明细账（星辰3.1以上版本支持）
-   请求方式：POST
-   请求地址：https://api.kingdee.com/jdy/v2/fi/assistdetail\_report

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### body参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| assist\_list | List <AssistList> | true | 辅助核算参数列表 |
| end\_period | string | true | 结束年度期间（例：“202405”） |
| start\_period | string | true | 开始年度期间（例：“202405”） |
| account\_id | List <string> | false | 科目id，与科目编码同时有值，以id为准 |
| account\_number | List <string> | false | 科目编码，与科目id同时有值,以id为准 |
| currency\_id | string | false | 币别id(0：综合本位币，-1：所有币别，默认0) |
| explanation | string | false | 摘要 |
| hide\_balance\_zero | string | false | 余额为0不显示（1：是，0：否，默认0） |
| hide\_no\_happen | integer | false | 无发生额且余额为0不显示（1：是，0：否，默认0） |
| hide\_total | string | false | 发生额为0不显示本期合计、本年累计（1：是，0：否，默认1） |
| page | string | false | 当前页（默认1，如page页超出最大页，则取最后一页） |
| page\_size | string | false | 每页显示条数，大于0，不传该字段表示不分页 |
| price\_scale | string | false | 单价精度（默认4位） |
| qty\_scale | string | false | 数量精度（默认2位） |

**AssistList**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| id | string | true | 过滤具体的辅助核算id，与number同时有值，以id为准；2）与number二选一必录 |
| number | string | true | 过滤具体的辅助核算编码：1）与id同时有值，以id为准；2）与id二选一必录 |
| type | string | true | 辅助核算类别id，通过【辅助核算类别列表】接口assist\_type\_list获取 |

#### 请求示例

```bash
curl --location --request  post 'https://api.kingdee.com/jdy/v2/fi/assistdetail_report' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com' \
--data-row ' { 	"account_id":[], 	"account_number":[], 	"assist_list":[ 		{ 			"id":"1", 			"number":"KH00010", 			"type":"f0001" 		} 	], 	"currency_id":"客户", 	"end_period":"202405", 	"explanation":"aa", 	"hide_balance_zero":"1", 	"hide_no_happen":1, 	"hide_total":"1", 	"page":"1", 	"page_size":"1", 	"price_scale":"1", 	"qty_scale":"1", 	"start_period":"202405" }'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| items | List <Items> | 数据行 |
| page\_count | string | 当前页返回行数 |
| total\_count | string | 科目总数 |

**Items**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| account\_name | string | 科目名称 |
| account\_number | string | 科目编码 |
| balance\_amount | string | 余额本位币金额 |
| balance\_amount\_for | string | 余额原币金额 |
| balance\_dc | string | 余额方向 |
| balance\_price | string | 余额单价 |
| balance\_qty | string | 余额数量 |
| credit\_amount | string | 贷方本位币金额 |
| credit\_amount\_for | string | 贷方原币金额 |
| credit\_price | string | 贷方单价 |
| credit\_qty | string | 贷方数量 |
| currency | string | 币别/汇率 |
| data\_type | string | 数据类型，0：期初数据，1：凭证数据，2：本期合计，3：本年累计 |
| date | string | 日期 |
| debit\_amount | string | 借方本位币金额 |
| debit\_amount\_for | string | 借方原币金额 |
| debit\_price | string | 借方单价 |
| debit\_qty | string | 借方数量 |
| explanation | string | 摘要 |
| id | string | 凭证id |
| type | string | 凭证字号 |
| unit | string | 单位 |
