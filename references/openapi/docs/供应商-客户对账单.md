# 供应商/客户对账单

### 基本信息

-   用途说明：获取供应商/客户 对账单
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/arap/reconciliation\_statement

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| end\_date | string | true | 结束日期（“yyyy-MM-dd”） |
| start\_date | string | true | 开始日期（“yyyy-MM-dd”） |
| dept\_ids | array | false | 部门id |
| emp\_ids | array | false | 业务员id |
| include\_oth | string | false | 是否包括其他应收 |
| include\_uncheck | string | false | 是否包括未审核 |
| is\_not\_show\_settle\_bill | string | false | 是否不显示已完全核销单 |
| is\_not\_show\_total | string | false | 是否不显示小计行 |
| is\_only\_show\_total | string | false | 是否仅显示小计行 |
| is\_show\_ls | string | false | 是否显示零售结存单 |
| is\_show\_settle\_src\_bill | string | false | 是否不显示已完全核销单 |
| item\_group\_ids | array | false | 客户/供应商 分类id |
| item\_ids | array | false | 客户/供应商 id |
| not\_view\_zero | string | false | 是否不显示余额为0 |
| report\_entity | string | false | ap\_statementrpt：供应商对账单（默认），ar\_statementrpt：客户对账单 |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/arap/reconciliation_statement?end_date=2022-01-01&start_date=2022-01-01&dept_ids=[]&emp_ids=[]&include_oth=true&include_uncheck=true&is_not_show_settle_bill=true&is_not_show_total=true&is_only_show_total=true&is_show_ls=true&is_show_settle_src_bill=true&item_group_ids=[]&item_ids=[]&not_view_zero=true&report_entity=100' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | string | 总条数 |
| current\_page\_size | integer | 当前页条数 |
| page | integer | 当前页码 |
| page\_size | integer | 每页条数 |
| rows | List <Row> | 数据行 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| account\_check | string | 对账状态 |
| amount | number | 单据金额 |
| amount\_for | number | 单据金额（本位币） |
| balance | number | 往来余额 |
| balance\_for | number | 往来余额（本位币） |
| bill\_id | string | 单据id |
| bill\_no | string | 单据编码 |
| bill\_type | string | 单据类型 |
| currency | string | 币别id |
| dis\_amount | number | 整单折扣额 |
| dis\_rate | string | 整单折扣率(%) |
| due\_date | string | 结算日期 |
| invoice\_status | string | 开票状态 |
| invoiced\_amount | number | 已开票金额 |
| item\_id | string | 客户/供应商 id |
| item\_name | string | 结算客户/供应商 名称 |
| item\_name\_biz | string | 客户/供应商 名称 |
| remark | string | 单据备注 |
| settle\_amount | number | 已核销金额 |
| settle\_amount\_for | number | 已核销金额（本位币） |
| settle\_src\_bill | string | 关联核销单据号 |
| settle\_src\_bill\_json | string | 关联单据Json |
| un\_settle\_amount | number | 待付款金额 |
| un\_settle\_amount\_for | number | 待付款金额（本位币） |
| voucher\_id | string | 凭证ID |
| voucher\_no | string | 凭证编码 |
| voucher\_period | string | 凭证期间 |
