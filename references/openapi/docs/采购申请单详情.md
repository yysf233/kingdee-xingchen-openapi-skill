# 采购申请单详情

### 基本信息

-   用途说明：采购申请单详情
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/pur\_request\_detail

```bash

--header 'X-Api-Auth-Version: 2.0' \

```
