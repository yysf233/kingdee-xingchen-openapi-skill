# 生产退料单保存

-   用途说明：生产退料单保存

-   请求地址：https://api.kingdee.com/jdy/v2/scm/inv\_pick\_ret

| errcode | integer | 返回码，成功时为0 |

| data | Data | 业务返回具体对象 |

| id\_number\_map | Map <string> | id与编码map |
