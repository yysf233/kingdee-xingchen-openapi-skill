# 销售出库单列表

### 基本信息

-   用途说明：销售出库单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/sal\_out\_bound

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| bill\_no | string | false | 单据编码 |
| bill\_status | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| create\_end\_time | string | false | 创建时间-结束时间的时间戳(毫秒) |
| create\_start\_time | string | false | 创建时间-开始时间的时间戳(毫秒) |
| creator\_id | array | false | 创建人ID |
| currency\_id | array | false | 币别 |
| customer\_id | array | false | 客户 |
| customer\_name | string | false | 客户名称 |
| customer\_number | string | false | 客户编码 |
| delivery\_status | string | false | 配送状态（所有：“”，全部配送：“A”，部分配送：“P”，未配送：“N”，无需配送：“C”,待配送（=部分配送+未配送）：“M”） |
| delivery\_type | string | false | 交货方式编码（所有：“”，物流发货：“JHFS01”，车辆配送：“JHFS03”，客户自提：“JHFS04”） |
| dept\_id | array | false | 部门ID |
| dept\_number | string | false | 部门编码 |
| emp\_id | array | false | 职员ID |
| emp\_number | string | false | 业务员编码 |
| end\_bill\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| filter\_zero | string | false | 过滤字段值为0记录：目前仅支持totalunsettleamount |
| material\_model | string | false | 商品规格型号 |
| material\_name | string | false | 商品名称 |
| material\_number | string | false | 商品编码 |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| payment\_flag | string | false | 是否同单收款（全部：-1，1：是，0：否） |
| settle\_status | string | false | 付款状态（所有：“”，全部收款：“C”，部分收款：“B”，未收款：“A”，待收款（=部分收款+未收款）：“M”） |
| show\_material\_total | string | false | 是否统计商品种类和数量到单据，默认false |
| src\_bill\_no | string | false | 源单单号 |
| start\_bill\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| top | string | false | 截取单据前top个商品数据返回 |
| total\_fields | string | false | 合计值字段，多个字段用英文逗号隔开，支持的字段：allamount（默认：价税合计）、totalunsettleamount（未收金额）、totalinsamount（收款合计）、totalamount（本次应收账款） |

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| headers | Header | 返回数据头信息 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_dis\_amount | string | 整单折扣额 |
| bill\_dis\_before\_amount | string | 单据整单折前价税合计 |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间 |
| custom\_field | Map <string> | 自定义字段[使用指南](https://open.jdy.com/#/files/api/detail?id=76567ff2a06311edaa4b3d71bf0fce53&noside=true) |
| customer\_id | string | 客户id |
| customer\_name | string | 客户名称 |
| customer\_number | string | 客户编码 |
| delivery\_status | string | 配送状态（所有：“”，全部配送：“A”，部分配送：“P”，未配送：“N”,无需配送：“C”） |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| express\_entity | List <ExpressEntity> | 物流信息 |
| id | string | 单据ID |
| ivc\_status | string | 单据开票状态\[1:未开票,2:部分开票,3:已开票\] |
| material\_entity | List <MaterialEntity> | 商品分录 |
| material\_group | integer | 单据商品种类 |
| material\_qty | string | 单据商品数量 |
| settle\_status | string | 付款状态（所有：“”，全部收款：“C”，部分收款：“B”，未收款：“A”） |
| total\_amount | string | 单据价税合计 |

**MaterialEntity**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| cost | string | 出库成本 |
| entry\_ivc\_status | string | 行开票状态：1-未开票；2-部分开票；3-全部开票；4-无需开票 |
| material\_id | string | 商品id |

**ExpressEntity**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| id | string | 分录id |
| delivery\_bill\_no | string | 物流单号 |
| delivery\_time | string | 发货时间 |
| express\_company\_id | string | 物流公司Id |
| express\_remark | string | 物流备注 |

**Header**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| total\_amount | string | 列表价税合计汇总，符合条件所有单据价税合计汇总，包含本页之外的数据 |
| total\_ins\_amount | string | 列表收款合计汇总，符合条件所有单据收款合计汇总，包含本页之外的数据 |
| total\_un\_settle\_amount | string | 列表未收金额汇总，符合条件所有单据未收金额汇总，包含本页之外的数据 |
