# 生产任务单保存

### 基本信息

-   用途说明：生产任务单保存
-   请求方式：POST
-   请求地址：https://api.kingdee.com/jdy/v2/pm/mo\_task\_bill

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| id\_number\_map | Map <string> | id与编码map |
| ids | List <string> | 单据id数组对象 |
