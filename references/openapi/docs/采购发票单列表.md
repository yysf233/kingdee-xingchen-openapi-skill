# 采购发票单列表

### 基本信息

-   用途说明：采购发票单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/pur\_invoice

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| account\_check | string | false | 对账状态 （所有：”“，已对账：”1“，未对账：“0”） |
| barcode | string | false | 商品条码 |
| begin\_time | string | false | 开始时间（格式：“时间戳”，-1表示不过滤），修改时间 |
| bill\_sp\_id | array | false | 表头仓位ID |
| bill\_status | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| bill\_stock\_id | array | false | 表头仓库ID |
| dept\_id | array | false | 部门ID |
| emp\_id | array | false | 职员ID |
| end\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| expire\_time | string | false | 结束时间（格式：“时间戳”，-1表示不过滤），修改时间 |
| hook\_status | string | false | 入库状态 （所有：”“，未入库：”0“，部分入库：“1”，全部入库：“2”） |
| ivc\_type | string | false | 发票类型 （所有：”“，普票：”1“，专票：“2”） |
| page | string | false | 当前页（默认1） |
| page\_size | string | false | 每页显示条数，（默认10，最大值100） |
| sp\_id | array | false | 仓位ID |
| start\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| stock\_id | array | false | 仓库ID |
| supplier\_id | array | false | 供应商ID |
| total\_fields | string | false | 合计值字段，多个字段用英文逗号隔开，支持的字段：totalunsettleamount（未收金额）、totalinsamount（收款合计）、totalamount（本次应收账款） |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/scm/pur_invoice?account_check=1&barcode=1&begin_time=1&bill_sp_id=[]&bill_status=C&bill_stock_id=[]&dept_id=[]&emp_id=[]&end_date=2020-02-22&expire_time=1&hook_status=1&ivc_type=1&page=1&page_size=1&sp_id=[]&start_date=2020-02-22&stock_id=[]&supplier_id=[]&total_fields=1' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| headers | Header | 返回数据头信息 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <InvoiceList> | 返回数据列表 |
| total\_page | integer | 总页数 |

**InvoiceList**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间 |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| id | string | 单据ID |
| remark | string | 备注 |
| settle\_status | string | 付款状态(全部收款：“C”，部分收款：“B”，未收款：“A”) |
| supplier\_id | string | 供应商id |
| supplier\_name | string | 供应商名称 |
| supplier\_number | string | 供应商编码 |

**Header**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| total\_amount | string | 列表价税合计汇总，符合条件所有单据价税合计汇总，包含本页之外的数据 |
| total\_ins\_amount | string | 列表收款合计汇总，符合条件所有单据收款合计汇总，包含本页之外的数据 |
| total\_un\_settle\_amount | string | 列表未收金额汇总，符合条件所有单据未收金额汇总，包含本页之外的数据 |
