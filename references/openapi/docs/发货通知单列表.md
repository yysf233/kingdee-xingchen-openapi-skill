# 发货通知单列表

### 基本信息

-   用途说明：发货通知单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/sal\_delivery\_notice

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| barcode | string | false | 商品条码 |
| begin\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| bill\_close\_state | string | false | 单据关闭状态（所有：“”，未关闭：“C”，已关闭：“S”,手动关闭: H） |
| bill\_no | string | false | 单据编码 |
| bill\_status | string | false | 单据状态，已审核：“C”，未审核：“Z” |
| customer\_group\_id | array | false | 客户分类id |
| dept\_ids | array | false | 部门id |
| emp\_ids | array | false | 业务员id |
| end\_bill\_date | string | false | 单据日期-结束日期（“yyyy-MM-dd”） |
| end\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| expire\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| io\_status | string | false | 执行状态（全部执行：“C”，部分执行：“Z”，未执行：“A”，待执行（=部分执行+未执行）：“B”） |
| page | string | false | 当前页号，默认为1 |
| page\_size | string | false | 每页显示条数，（默认10，最大值100） |
| real\_io\_status | string | false | 出库状态（全部出库：“C”，部分出库：“Z”，未出库：“A”，待出库（=部分出库+未出库）：“B”） |
| show\_only\_push\_move | string | false | 是否仅返回符合单据下推的数据(未关闭，未执行或部分执行) |
| start\_bill\_date | string | false | 单据日期-开始日期（“yyyy-MM-dd”） |
| start\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| stock\_ids | array | false | 仓库ID集合 |
| total\_fields | string | false | 自定义合计字段，多个字段用英文逗号隔开，目前支持的字段：totalmount（列表价税合计汇总） |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/scm/sal_delivery_notice?barcode=0045&begin_date=2022-05-02&bill_close_state=C&bill_no=001&bill_status=Z&customer_group_id=[]&dept_ids=[]&emp_ids=[]&end_bill_date=2022-01-01&end_date=2022-05-02&expire_date=2022-05-02&io_status=C&page=1&page_size=1&real_io_status=C&show_only_push_move=false&start_bill_date=2022-01-01&start_date=2022-05-02&stock_ids=[]&total_fields=allamount' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| headers | Header | 返回数据头信息（查询参数传递total\_fields） |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_close\_state | string | 单据关闭状态（未关闭：“C”，已关闭：“S”,手动关闭: H） |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_dis\_amount | string | 折扣额 |
| bill\_dis\_before\_amount | string | 单据整单折前价税合计 |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间 |
| customer\_id | string | 客户id |
| customer\_name | string | 客户名称 |
| customer\_number | string | 客户编码 |
| delivery\_type | string | 交货方式编码（物流发货：“JHFS01”，车辆配送：“JHFS03”，客户自提：“JHFS04”） |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| id | string | 单据ID |
| io\_status | string | 执行状态（全部执行：“C”，部分执行：“Z”，未执行：“A”，待执行（=部分执行+未执行）：“B”） |
| real\_io\_status | string | 出库状态（全部出库：“C”，部分出库：“Z”，未出库：“A”，待出库（=部分出库+未出库）：“B”） |
| total\_amount | string | 单据价税合计 |

**Header**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| total\_amount | string | 列表价税合计汇总，符合条件所有单据价税合计汇总，包含本页之外的数据 |
