# 变动项类型

### 变动项类型

| altert\_ype值 | alter\_type含义 | alter\_value/alter\_value\_list类型 | 参数说明 |
| --- | --- | --- | --- |
| txtdept | 使用部门 | array<data.txtdept> | 变动后的使用部门对象 |
| originalvalue | 原值 | array<data.originalvalue> | 变动后的原值对象 |
| txtexpenseacct | 折旧费用分配科目 | array<data.acctList> | 变动后的多科目对象 |
| txtoppositeacct | 减值准备对方科目 | array<data.acctList> | 变动后的多科目对象 |
| group | 资产类别 | string | 资产类别id |
| name | 资产名称 | string | 资产名称 |
| qty | 数量 | string | 数量 |
| location | 存放地点 | string | 存放地点id |
| usestatus | 使用状态 | string | 使用状态id |
| emp | 使用人 | string | 职员id |
| deprmethod | 折旧方法 | string | 年限平均法：“10”；双倍余额递减法：“20”；年数总和法：“30”；工作量法：“35”；不折旧：“40”； |
| deprtype | 折旧依据string | string | 按原值：“1”；按净值：“0” |
| tax | 税额 | String | 金额 |
| decprevalue | 减值准备 | string | 金额 |
| accumdepr | 累计折旧 | string | 金额 |
| residuevalue | 净残值 | string | 金额 |
| assetacctid | 固定折旧科目 | string | 科目id |
| depracctid | 累计折旧科目 | string | 科目id |
| taxacctid | 税金科目 | string | 科目id |
| decpreacctid | 减值准备科目 | string | 科目id |
| alteracctid | 资产变动对方科目 | string | 科目id |
| maxdeprperiods | 预计折旧期间数 | string |  |

#### 变动后的使用部门对象 txtdept

| 参数名称 | 类型 | 必选 | 说明 |
| --- | --- | --- | --- |
| dept\_id | string | true | 变动后的部门id |
| percent | string | true | 部门占比（%）；范围(0,100\]， 所有percent合计为100 |

#### 变动后的原值对象 originalvalue

| 参数名称 | 类型 | 必选 | 说明 |
| --- | --- | --- | --- |
| currency\_id | string | false | 币别id，本位币可以不传 |
| exchange\_rate | string | false | 币别汇率（数值），不赋值默认为1 |
| amount\_for | string | false | 原币金额（数值），不赋值默认为amount值，不会检测汇率和原币、本位币金额是否一致 |
| amount | string | true | 本位币金额（数值，按系统本位币精度） |

#### 变动后的多科目对象 acctList （适用于txtexpenseacct、txtoppositeacct）

| 参数名称 | 类型 | 必选 | 说明 |
| --- | --- | --- | --- |
| dept\_id | string | true | 部门id；如果没有变动部门，取原部门id；否则取变动后的部门id |
| account\_id | string | true | 折旧分配科目id |
| assist | array<data.assist> | false | 科目对应的核算项目，科目启用了必录核算项目是必录 |
| percent | string | true | 同一部门的分配比例，范围(0,100\]， 同一部门占比合计为100 |

#### 核算项目列表对象 （assist）

| 参数名称 | 类型 | 必选 | 说明 |
| --- | --- | --- | --- |
| type | string | true | 辅助核算类别(客户-bd\_customer，供应商-bd\_supplier， 商品-bd\_material，部门-bd\_department，职员-bd\_employee，仓库-bd\_store)， 辅助资料-bd\_auxinfo |
| bd\_aux\_info\_type\_id | string | false | 辅助资料类别id，**当type为bd\_auxinfo时必填** |
| id | string | true | type类别下的基础资料id，例如type=“bd\_customer”，id值为客户id |
