# 盘点任务单保存

### 基本信息

-   用途说明：盘点任务单保存
-   请求方式：POST
-   请求地址：https://api.kingdee.com/jdy/v2/scm/inv\_check\_task

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### body参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| name | string | true | 任务名称 |
| stock\_id | string | true | 仓库id |
| check\_batch\_kf\_period | boolean | false | 按批次保质期盘点,默认false |
| check\_date | string | false | 盘点日期，指定日期盘点时才有 |
| check\_model | string | false | 盘点模式(默认:单人盘点:“1”,多人盘点:“2”) |
| check\_scope | string | false | 盘点范围（默认:全盘:“1”,抽盘:“2”） |
| check\_serial | boolean | false | 按序列号盘点,默认false |
| check\_type | string | false | 盘点类型(即时库存盘点:“1”,指定日期盘点:“2”) |
| checker\_number | string | false | 盘点人(值为职员的编码,多个之间用英文逗号分隔开) |
| create\_type | string | false | 商品清单展示(默认:空白盘点单:“1”,显示商品清单:“2”) |
| id | string | false | 任务ID,默认为0 |
| include\_forbidden | boolean | false | 包含禁用商品，默认false |
| include\_zero | boolean | false | 包含零库存商品，默认false |
| lock\_status | boolean | false | 是否锁盘,默认false |
| material\_brand\_number | string | false | 商品品牌(值为商品品牌的编码,多个之间用英文逗号分隔开) |
| material\_category\_number | string | false | 商品分(值为商品分类的编码,多个之间用英文逗号分隔开) |
| material\_number | string | false | 商品（值为商品的编码，多个之间用英文逗号分隔开） |
| material\_source | string | false | 商品来源(默认:在盘点仓库中有交易记录的商品:“2”,商品资料中所有商品:“1”) |
| remark | string | false | 备注 |
| sp\_number | string | false | 仓位(值为仓位的编码,多个之间用英文逗号分隔开) |

#### 请求示例

```bash
curl --location --request  post 'https://api.kingdee.com/jdy/v2/scm/inv_check_task' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com' \
--data-row ' { 	"check_batch_kf_period":1423432, 	"check_date":"2020-10-14", 	"check_model":"1", 	"check_scope":"1", 	"check_serial":1423432, 	"check_type":"1", 	"checker_number":"DK1", 	"create_type":"1", 	"id":null, 	"include_forbidden":false, 	"include_zero":false, 	"lock_status":true, 	"material_brand_number":"143", 	"material_category_number":"1324", 	"material_number":"", 	"material_source":"1", 	"name":null, 	"remark":"14", 	"sp_number":"1423432", 	"stock_id":"321424" }'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| ids | List <string> | 成功的id |
| check\_bill\_id | string | 盘点单ID |
| check\_bill\_no | string | 盘点单编码 |
