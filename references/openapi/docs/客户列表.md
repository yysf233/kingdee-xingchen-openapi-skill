# 客户列表

### 基本信息

-   用途说明：获取客户列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/bd/customer

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| c\_level | string | false | 价格等级id |
| contact\_person\_main | string | false | 主要联系人名称 |
| create\_end\_time | string | false | 创建时间-结束时间的时间戳(毫秒) |
| create\_start\_time | string | false | 创建时间-开始时间的时间戳(毫秒) |
| customer\_maturity | string | false | 成熟度id |
| enable | integer | false | 状态(1:启用 0:禁用 -1：不限) |
| group | array | false | 类别id |
| is\_data\_perm | boolean | false | 是否添加数据权限校验，默认false |
| mobile\_main | string | false | 主要联系人电话 |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| order\_by | string | false | 排序；编码升序：number asc（默认），编码降序：number desc，名称升序：name asc，名称降序：name desc |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| sale\_dept\_id | string | false | 部门id |
| saler\_id | string | false | 业务员id |
| search | string | false | 模糊搜索-名称 |
| show\_alarm | boolean | false | 是否显示客户预警信息；默认false-不显示 |
| show\_business\_time | boolean | false | 是否显示最近交易时间；默认false-不显示 |
| show\_contact\_detail | boolean | false | 是否显示客户联系人信息；默认false-不显示 |
| show\_debt | boolean | false | 是否显示客户欠款 |
| show\_trace\_info | boolean | false | 是否显示最近一条跟进记录 |
| show\_un\_visit\_days | boolean | false | 是否显示未拜访天数；默认false-不显示 |
| trace\_end\_date | string | false | 跟进结束时间（格式：“yyyy-MM-dd”） |
| trace\_start\_date | string | false | 跟进开始时间（格式：“yyyy-MM-dd”） |

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/bd/customer?c_level=1&contact_person_main=001&create_end_time=1234567890123&create_start_time=1234567890123&customer_maturity=1&enable=1&group=[]&is_data_perm=true&mobile_main=138&modify_end_time=1234567890123&modify_start_time=1234567890123&order_by=name desc&page=1&page_size=100&sale_dept_id=1&saler_id=1&search=苹果&show_alarm=false&show_business_time=false&show_contact_detail=false&show_debt=true&show_trace_info=001&show_un_visit_days=false&trace_end_date=2021-11-11&trace_start_date=2021-11-11' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \

--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| current\_page | integer | 当前页码 |
| count | string | 总条数 |
| current\_page\_size | integer | 当前页条数 |
| page | integer | 当前页码 |
| page\_size | integer | 每页条数 |
| rows | List <Row> | 数据行 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| id | string | 客户id |
| c\_level\_id | string | 等级对象id |
| enable | string | 状态（1-可用 0-不可用） |
| group\_name | string | 类别id |
| name | string | 名称 |
| number | string | 编码 |
| remark | string | 备注 |
