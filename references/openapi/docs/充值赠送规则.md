# 充值赠送规则

### 基本信息

-   用途说明：会员充值赠送规则列表查询
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/ls/mall\_recharge\_rule

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| member\_id | string | true | 会员id |
| member\_card\_id | string | false | 会员卡id |
| store\_id | string | false | 门店id |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/ls/mall_recharge_rule?member_id=1&member_card_id=1&store_id=1' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| rows | List <Row> |  |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| coupon\_list | List <Coupon> | 赠送券列表 |
| donate\_amount | string | 赠送金额 |
| id | string | id |
| material\_list | List <Material> | 商品列表 |
| material\_set | boolean | 是否指定商品数量 |
| material\_value | string | 指定商品种类数（X种 |
| recharge\_amount | string | 充值金额 |
| rule\_id | string | 充值规则id |
| to\_amount | boolean | 赠送余额 |
| to\_coupon | boolean | 赠送优惠券 |
| to\_material | boolean | 赠送商品 |
| to\_member\_point | string | 赠送积分 |
| to\_point | boolean | 赠送积分 |

**Material**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| aux\_comb\_id | string | 辅助属性组合id |
| aux\_comb\_name | string | 辅助属性组合名称 |
| aux\_prop\_id | string | 辅助属性id（进销存辅助属性id实际值为零售的skuId） |
| bd\_id | string | 赠送id |
| bd\_name | string | 商品名称 |
| img\_url | string | 商品图片 |
| is\_aux | string | 是否批次商品（0：否，1：是） |
| is\_batch | string | 是否批次商品（0：否，1：是） |
| is\_selected | boolean | 是否默认选项 |
| is\_serial | string | 是否序列号商品（0：否，1：是） |
| material\_id | string | 商品id |
| material\_name | string | 商品名称 |
| num | string | 赠送商品数量 |
| price | string | 零售价 |
| sku\_entry\_id | string | sku表id |
| small\_img\_url | string | 商品缩略图 |
| unit | string | 单位id |
| unit\_name | string | 单位名称 |

**Coupon**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| begin\_date | string | 固定时间赠送的开始时间 |
| coupon\_schema\_id | string | 优惠券方案id |
| coupon\_schema\_name | string | 优惠券方案名称 |
| delay\_days | string | 延迟赠送的天数 |
| effect\_range | string | 有效时间 |
| effect\_type | string | 生效方式 |
| effect\_type\_desc | string | 生效方式描述 |
| end\_date | string | 固定时间赠送的结束时间 |
| given\_type | string | 赠送方式 |
| given\_type\_desc | string | 赠送方式描述 |
| interval | string | 赠送间隔次数 |
| per\_qty | string | 每次赠送数量 |
| qty | string | 总数量数量 |
| terms | string | 赠送次数 |
