# 拣验货单列表

### 基本信息

-   用途说明：拣验货单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/inv\_pick\_inspect

| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |

| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| begin\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| bill\_status | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| dept\_id | array | false | 领料部门ID |
| emp\_id | array | false | 领料人ID |
| end\_bill\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| end\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| end\_time | string | false | 结束时间（格式：“时间戳”，-1表示不过滤） |
| expire\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| page | integer | false | 当前页（默认1） |
| page\_size | integer | false | 每页显示条数，（默认10，最大值100） |
| pick\_status | string | false | 拣验货状态，1：未开始；2：拣验中；3：拣验完成，4：待拣验（包括未开始和拣验中） |
| src\_bill\_no | string | false | 源单编码 |
| src\_bill\_type | string | false | 源单类型，sal\_bill\_order：销售订单；pur\_bill\_order：采购订单 |
| src\_customer\_id | array | false | 源单客户ID |
| src\_supplier\_id | array | false | 源单供应商ID |
| start\_bill\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| start\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| start\_time | string | false | 开始时间（格式：“时间戳”，-1表示不过滤） |
| total\_fields | string | false | 自定义合计字段，合计的值为符合条件的合计，和分页无关，多个字段用英文逗号隔开，如：cost |
| trans\_type | string | false | 任务类型,1：一单一拣；2：合并拣货 |

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| cur\_page\_size | integer | 当前页大小 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| id | string | 单据ID |
| pick\_status | string | 拣验货状态，1：未开始；2：拣验中；3：拣验完成，4：待拣验（包括未开始和拣验中） |
| remark | string | 备注 |
| src\_bill\_type | string | 源单类型，sal\_bill\_order：销售订单；pur\_bill\_order：采购订单 |
| trans\_type | string | 任务类型,1：一单一拣；2：合并拣货 |
