# 销售发票单列表

### 基本信息

-   用途说明：销售发票单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/sal\_invoice\_list

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| end\_date | string | true | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| start\_date | string | true | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），创建日期 |
| account\_check | string | false | 对账状态 （所有：”“，已对账：”1“，未对账：“0”） |
| barcode | string | false | 条形码 |
| begin\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| bills\_tatus | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| customer\_group\_id | array | false | 客户分类 |
| customer\_id | array | false | 客户 |
| dept\_id | array | false | 部门ID |
| emp\_id | array | false | 职员ID |
| end\_bill\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| expire\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），修改日期 |
| hook\_status | string | false | 出库核销状态 （所有：”“，未核销：”0“，部分核销：“1”，已核销：“2”） |
| invoice\_type | string | false | 发票类型 （所有：”“，普票：”1“，专票：“2”） |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| payment\_flag | string | false | 是否同单收款（全部：-1，1：是，0：否） |
| settle\_status | string | false | 付款状态（所有：“”，全部收款：“C”，部分收款：“B”，未收款：“A”，待收款（=部分收款+未收款）：“M”） |
| show\_material\_total | string | false | 是否统计商品种类和数量到单据，默认false |
| src\_bill\_no | string | false | 源单编号 |
| start\_bill\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| stock\_id | array | false | 仓库 |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/scm/sal_invoice_list?end_date=2022-01-01&start_date=2022-01-01&account_check=0&barcode=0045&begin_date=2022-05-02&bills_tatus=z&customer_group_id=[]&customer_id=[]&dept_id=002&emp_id=001&end_bill_date=2022-01-01&expire_date=2022-05-02&hook_status=0&invoice_type=1&page=1&page_size=100&payment_flag=1&settle_status=C&show_material_total=false&src_bill_no=001&start_bill_date=2022-01-01&stock_id=[]' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | string | 总条数 |
| current\_page\_size | integer | 当前页条数 |
| page | integer | 当前页码 |
| page\_size | integer | 每页条数 |
| rows | List <Row> | 数据行 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期 |
| id | string | id |
| bill\_dis\_amount | number | 整单折扣额 |
| bill\_no | string | 单据编号 |
| bill\_status | string | 单据状态（所有：“”，已审核：“C”，未审核：“A”） |
| create\_time | string | 创建时间 |
| customer\_id | string | 客户id |
| customer\_name | string | 客户名称 |
| customer\_number | string | 客户编码 |
| db\_modify\_time | string | 修改时间 |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| emp\_number | string | 业务员编码 |
| material\_group | string | 单据商品种类 |
| material\_qty | string | 单据商品数量 |
| settle\_status | string | 本单核销状态 |
| total\_amount | string | 本单合计金额 |
