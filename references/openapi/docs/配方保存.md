# 配方保存

### 基本信息

-   用途说明：配方新增及修改
-   请求方式：POST
-   请求地址：https://api.kingdee.com/jdy/v2/ls/formula

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### body参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| data | FormulaSavaReqData | true | 保存入参 |

**FormulaSavaReqData**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| material\_id | string | true | 商品id |
| material\_list | List <MaterialRow> | true | 原料商品列表 |
| name | string | true | 配方名 |
| unit\_id | string | true | 主商品单位id |
| aux\_prop\_id | string | false | 主商品辅助属性id，如果商品启用辅助属性管理，则必填 |
| id | string | false | 配方id,新增不填，更新时必填（id为空时代表新增） |
| remark | string | false | 备注 |
| store\_list | List <StoreRow> | false | 门店信息列表 |
| suit\_store\_type | string | false | 全部门店=1 |

**StoreRow**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| type | string | false | 类型:1=门店；2=标签 |
| type\_id | string | false | 门店ID |

**MaterialRow**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| material\_id | string | true | 原料商品ID |
| qty | number | true | 原料商品消耗量 |
| unit\_id | string | true | 商品单位ID |
| allow\_replace | boolean | false | 原料商品是否可以替换.默认=false |
| aux\_prop\_id | string | false | 商品辅助属性id，如果商品启用辅助属性管理，则必填 |
| other\_material\_list | List <OtherMaterialRow> | false | 替换商品列表 |

**OtherMaterialRow**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| material\_id | string | true | 原料替换商品ID |
| qty | number | true | 原料替换商品消耗量 |
| unit\_id | string | true | 原料替换商品单位ID |
| aux\_prop\_id | string | false | 原料替换商品辅助属性id，如果商品启用辅助属性管理，则必填 |

#### 请求示例

```bash
curl --location --request  post 'https://api.kingdee.com/jdy/v2/ls/formula' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com' \
--data-row ' { 	"data":{ 		"aux_prop_id":"697140092908614656", 		"id":"001", 		"material_id":"697140092908614656", 		"material_list":[ 			{ 				"allow_replace":null, 				"aux_prop_id":"697140092908614656", 				"material_id":"697140092908614656", 				"other_material_list":[ 					{ 						"aux_prop_id":"697140092908614656", 						"material_id":"697140092908614656", 						"qty":1, 						"unit_id":"1" 					} 				], 				"qty":1, 				"unit_id":"1" 			} 		], 		"name":"1", 		"remark":"备注", 		"store_list":[ 			{ 				"type":"1", 				"type_id":"1" 			} 		], 		"suit_store_type":"1", 		"unit_id":"697140092908614656" 	} }'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| id\_number\_map | Map <string> | id与编码map |
| ids | List <string> | 单据id数组对象 |

#### 响应示例

```json
{
  "data": {
    "id_number_map": {},
    "ids": [
      "1"
    ]
  },
  "description": "success",
  "errcode": 0
}
```
