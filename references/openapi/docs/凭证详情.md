# 凭证详情

### 基本信息

-   用途说明：获取凭证详情
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/fi/voucher\_detail

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| id | string | false | （id和number二选一） |
| number | string | false | 编码（id与number二选一） |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/fi/voucher_detail?id=1&number=001' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_status | string | 状态,暂存：“A”，保存：“B”，已审核：“C”，已作废：“D” |
| credit\_total | string | 贷方总金额 |
| date | string | 凭证日期 |
| debit\_total | string | 借方总金额 |
| group\_id | string | 凭证字id |
| group\_name | string | 凭证字名称 |
| group\_number | string | 凭证字编码 |
| have\_reverse | boolean | 是否已被冲销 |
| id | string | 凭证id |
| is\_reverse | boolean | 是否冲销凭证 |
| number | string | 凭证号 |
| period | string | 凭证期间 |
| voucher\_no | string | 凭证字号 |
| entry\_list | List <Entry> | 凭证分录 |
| src\_bill | List <SourceBill> | 凭证关联的源单信息 |

**SourceBill**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| src\_bill\_no | string | 业务单据number |
| src\_bill\_id | string | 业务单据id |
| src\_bill\_type | string | 业务单据类型详见[基础资料及单据类型](https://open.jdy.com/#/files/api/detail?index=3&categrayId=3cc8ee9a663e11eda5c84b5d383a2b93&id=dd7ee492712611eda0b375f5553347df) |

**Entry**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| account\_id | string | 科目 id |
| account\_name | string | 科目 名称 |
| account\_number | string | 科目 编码 |
| amount\_for | string | 金额（本位币） |
| assist | List <Assist> | 期间 |
| credit\_amount | string | 贷方金额 |
| currency\_id | string | 币别id |
| currency\_name | string | 币别名称 |
| currency\_number | string | 币别编码 |
| dc | string | 借贷方向（1：借，-1：贷） |
| debit\_amount | string | 借方金额 |
| explanation | string | 摘要 |
| id | string | 分录ID |
| measure\_unit\_id | string | 单位id |
| measure\_unit\_name | string | 单位名称 |
| measure\_unit\_number | string | 单位编码 |
| price | string | 单价 |
| quantity | string | 数量 |
| seq | string | 分录序号 |

**Assist**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| type | string | 辅助核算类别 |
| bd\_aux\_info\_type\_id | string | 当type为bd\_auxinfo时,辅助资料类别id |
| bd\_auxinfo\_type\_name | string | 当type为bd\_auxinfo时,辅助资料类别名称 |
| id | string | 辅助核算id（id与number二选一） |
| name | string | 辅助核算名称 |
| number | string | 辅助核算编码 |
| type\_name | string | 核算类别名称 |
