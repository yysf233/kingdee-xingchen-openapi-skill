# 手工开票详情

### 基本信息

-   用途说明：手工开票详情
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/fp/invoice\_kp\_detail

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| serial\_number | string | true | 流水号 |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/fp/invoice_kp_detail?serial_number=1' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| amt\_info | AmtInfo | 金额信息 |
| buyer\_info | BuyerInfo | 购方信息 |
| id | string | 发票id |
| other\_info | OtherInfo | 其他信息 |
| seller\_info | SellerInfo | 销方信息 |
| download\_url | string | 下载地址 |
| fail\_reason | string | 失败原因（status =2 时返回） |
| invoice\_status | string | 发票状态(1:开票中、2:失败、3:正常、4:红冲、5:作废) |
| invoice\_type | string | 发票类型 发票类型：(1:纸质专用发票、2：纸质普通发票、3：电子普通发票、4：电子专用发票、5：数电普通发票、6：数电专用发票、7：数电纸质普通发票、8：数电纸质专用发票) |
| serial\_number | string | 流水号。需要保证唯一长度小于50，未传递时由星辰生成。 |
| special\_type | string | 开具数电发票—特殊业务时传递。（货物运输服务：2） |
| transport\_info | List <TransportInfo> | 货物运输服务信息（货物运输服务业务必传） |
| invoice\_code | string | 发票代码 |
| invoice\_number | string | 发票号码 |

**TransportInfo**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| end\_place | string | 目的地 |
| license\_plate | string | 运输牌号 |
| start\_place | string | 起运地 |
| transport\_goods | string | 运输货物名称 |
| transport\_type | string | 运输工具种类(1：铁路运输、2：水路运输、3：公路运输、4：航空运输、5：管道运输 、6：其它运输工具) |

**SellerInfo**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| address\_phone | string | 销方地址电话 [敏感数据解密](https://open.jdy.com/#/files/api/detail?id=d7046d818dc111ee860d0fedad361070&noside=true) |
| bank | string | 开户银行 |
| name | string | 名称 |
| tax\_no | string | 纳税人识别号 |
| account | string | 银行账号 [敏感数据解密](https://open.jdy.com/#/files/api/detail?id=d7046d818dc111ee860d0fedad361070&noside=true) |

**OtherInfo**

| address\_phone | string | 销方地址电话 [敏感数据解密](https://open.jdy.com/#/files/api/detail?id=d7046d818dc111ee860d0fedad361070&noside=true) |
