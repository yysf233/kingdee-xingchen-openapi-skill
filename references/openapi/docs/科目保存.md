# 科目保存

### 基本信息

-   用途说明：科目新增及修改。审核、删除等详见[通用操作接口](https://open.jdy.com/#/files/api/detail?index=2&categrayId=3cc8ee9a663e11eda5c84b5d383a2b93&id=9e804b8c712511eda0b39f724d124b07)
-   请求方式：POST
-   请求地址：https://api.kingdee.com/jdy/v2/fi/account

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### body参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| items | List <AccountSaveDto> | false | 列表 |

**AccountSaveDto**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| account\_type\_id | string | true | 科目类别 id （只能填level为2的科目类别） |
| dc | string | true | 借贷方向 1借方 -1贷方 |
| level | string | true | 科目级次（1-n）一级科目保存时，level=1字段必填 |
| name | string | true | 科目名称 |
| number | string | true | 科目编码（编码要符合科目级次规则4-2-2-2，如1901属于一级科目编码，190101属于二级科目编码，以此类推） |
| check\_item\_entry | List <AccountCheckItemEntry> | false | 辅助核算分录，is\_assist为true时必填 |
| currency\_ids | string | false | 指定币别id，多个币别id用逗号隔 |
| enable | string | false | 启用,1-是 0-否 |
| help\_code | string | false | 助记码 |
| id | string | false | 新增不填，更新时必填（id为空时代表新增单据） |
| is\_all\_currency | boolean | false | 是否所有币别，默认false |
| is\_assist | boolean | false | 是否辅助核算 默认false |
| is\_bank | boolean | false | 是否银行科目1-是 0-否 |
| is\_cash | boolean | false | 是否现金科目1-是 0-否 |
| is\_cash\_equivalent | boolean | false | 是否现金等价物1-是 0-否 |
| is\_change\_currency | boolean | false | 是否期末调汇，默认false |
| is\_currency | boolean | false | 是否外币核算1-是 0-否 |
| is\_qty | boolean | false | 是否数量核算1-是 0-否 |
| measure\_unit\_id | string | false | 计量单位 |
| parent\_id | string | false | 上级科目 id |

**AccountCheckItemEntry**

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| bd\_aux\_info\_type\_id | string | false | 当type为bd\_auxinfo时，辅助资料类别id |
| id | string | false | 分录id |
| is\_require | boolean | false | 是否必录 |
| seq | string | false | 分录序号 |

#### 请求示例

```bash
curl --location --request  post 'https://api.kingdee.com/jdy/v2/fi/account' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com' \
--data-row ' { 	"items":[ 		{ 			"account_type_id":"1", 			"check_item_entry":[ 				{ 					"bd_aux_info_type_id":"1", 					"id":"1", 					"is_require":true, 					"seq":"1" 				} 			], 			"currency_ids":"[]", 			"dc":"1", 			"enable":"1", 			"help_code":"助记码", 			"id":"1", 			"is_all_currency":false, 			"is_assist":false, 			"is_bank":false, 			"is_cash":false, 			"is_cash_equivalent":false, 			"is_change_currency":false, 			"is_currency":false, 			"is_qty":false, 			"level":"级次", 			"measure_unit_id":"1", 			"name":"xxx", 			"number":"190101", 			"parent_id":"1" 		} 	] }'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| rows | List <SaveReply> | 保存单据返回数组 |

**SaveReply**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| id\_number\_map | Map <string> | id与编码map |
| ids | List <string> | 单据id数组对象 |

#### 响应示例

```json
{
  "data": {
    "rows": [
      {
        "id_number_map": {},
        "ids": [
          "1"
        ]
      }
    ]
  },
  "description": "success",
  "errcode": 0
}
```
