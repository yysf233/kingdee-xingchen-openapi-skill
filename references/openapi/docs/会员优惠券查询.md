# 会员优惠券查询

### 基本信息

-   用途说明：会员优惠券查询
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/ls/coupons

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| member\_id | string | true | 会员ID |
| cashier\_id | string | false | 收银员ID,可为空 |
| channel | string | false | 优惠券来源 |
| end\_time | string | false | 更新时间范围结束 |
| id | string | false | 编码支持ID |
| is\_effect | boolean | false | true 获取有效期内的 false 获取有效期外的 |
| mobile | string | false | 编码手机号 |
| page | string | false | 页数 默认为1 |
| page\_size | string | false | 每页记录数，默认为20 |
| schema\_id | string | false | 方案Id |
| start\_time | string | false | 更新时间范围起始 |
| verify\_stat | string | false | 是否可核销 1-是 2-否 为空则不过滤 |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/ls/coupons?member_id=1&cashier_id=1&channel=1&end_time=2022-05-01&id=1&is_effect=true&mobile=13311223344&page=1&page_size=20&schema_id=1&start_time=2022-05-01&verify_stat=1' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总条数 |
| cur\_page\_size | integer | 当前页条数 |
| page | integer | 当前页 |
| page\_size | integer | 每页条数 |
| rows | List <Row> | 数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_no | string | 零售单号 |
| channel | string | 优惠券来源(1-扫码领用；2-促销赠送；3-转增；4-购买；5-支付有礼；6-入门有礼；7-签到有礼) |
| comment | string | 优惠券说明 |
| content\_tip | string | 优惠内容 |
| coupon\_type | string | 优惠券方案类型(1-代金券；2-折扣券；3-抵扣券；4-多次核销代金券；5-多次核销抵扣券) |
| coupons\_id | string | 优惠券ID |
| coupons\_no | string | 优惠券编号 |
| coupons\_type | string | 优惠券类型(1-电子优惠券；2-纸质优惠券；3-付费电子券) |
| create\_time | string | 生成时间 |
| effect\_time | string | 生效时间 |
| expire\_time | string | 失效时间 |
| member\_card | MemberCard | 会员卡信息 |
| modify\_time | string | 业务修改时间 |
| schema\_id | string | 方案Id |
| schema\_name | string | 方案名称 |
| status | string | 优惠券状态 1:未到有效期 2：可使用 3：已过期 |
| suit\_material\_entity | string | 适用商品类型(1-全部商品；2-全部商品+排除；3-指定商品) |
| suit\_type\_store | string | 适用门店类型(1-全部门店；2-全部门店+排除；3-指定门店) |
| threshold | string | 使用门槛 |
| threshold\_value | string | 适用门槛阈值 |
| total\_amt | string | 优惠券金额 |
| usable\_amt | string | 优惠券可用金额 |
| value | string | 优惠券方案面值 |
| write\_off\_stat | string | 核销状态(1-可核销；2-不可核销) |

**MemberCard**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| card\_no | string | 会员卡卡号 |
| id | string | 会员卡Id |
