# 应收账款汇总表

### 基本信息

-   用途说明：获取应收账款汇总表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/arap/ar\_summary\_report

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| end\_date | string | true | 结束日期（“yyyy-MM-dd”） |
| start\_date | string | true | 开始日期（“yyyy-MM-dd”） |
| customer\_group\_ids | array | false | 客户类别id |
| customer\_ids | array | false | 客户id |
| dept\_ids | array | false | 部门id |
| emp\_ids | array | false | 业务员id |
| include\_oth | string | false | 是否包括其他应收 |
| include\_uncheck | string | false | 是否包括未审核 |
| is\_foreigncur | string | false | 是否启用了外币，预留字段，没有传值或者设置时默认为false |
| isshowls | string | false | 是否显示零售结存单 |
| not\_invoiced | boolean | false | 是否显示已出库/退货未开票,默认为false，该字段仅旗舰版支持 |
| not\_view\_no\_happen | string | false | 是否不显示无发生额 |
| not\_view\_zero | string | false | 是否不显示余额为0 |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| search | string | false | 模糊搜索，支持供应商名称，联系人，联系方式 |
| summary\_type | string | false | 汇总方式：客户item、业务员emp、部门dept、客户分类group |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/arap/ar_summary_report?end_date=2022-01-01&start_date=2022-01-01&customer_group_ids=[]&customer_ids=[]&dept_ids=[]&emp_ids=[]&include_oth=true&include_uncheck=true&is_foreigncur=false&isshowls=false&not_invoiced=false&not_view_no_happen=true&not_view_zero=true&page=1&page_size=100&search=001&summary_type=item' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | string | 总条数 |
| current\_page\_size | integer | 当前页条数 |
| page | integer | 当前页码 |
| page\_size | integer | 每页条数 |
| rows | List <Row> | 数据行 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| balance\_for | string | 余额-应收 |
| customer\_group\_id | string | 客户分类id |
| customer\_group\_name | string | 客户分类名称 |
| customer\_id | string | 客户id |
| customer\_name | string | 客户名称 |
| dept\_id | string | 部门id |
| dept\_name | string | 部门名称 |
| emp\_id | string | 业务员id |
| emp\_name | string | 业务员名称 |
| oth\_balance\_for | string | 其他应收余额 |
| pre\_balance\_for | string | 预收款余额 |
| rp\_scm\_amount\_for | string | 收销售款 |
| scm\_balance\_for | string | 销售应收余额 |
