# 科目余额表

### 基本信息

-   用途说明：查询科目余额表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/fi/account\_balance\_report

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| end\_Period | string | true | 结束年度期间 |
| start\_period | string | true | 开始年度期间 |
| currency\_id | string | false | 币别id（0：综合本位币，-1：所有币别，默认0） |
| page | integer | false | 当前页（默认1，如page页超出最大页，则取最后一页） |
| page\_size | integer | false | 每页显示条数，大于0，不传该字段表示不分页 |
| search | string | false | 模糊搜索，支持科目名称、科目编码、科目助记码 |
| show\_expand\_item | integer | false | 显示辅助核算（1：显示，0：不显示 默认1） |
| show\_sum\_acct\_type | integer | false | 显示科目类别合计（1：显示，0：不显示 默认0） |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/fi/account_balance_report?end_Period=201903&start_period=201903&currency_id=1&page=1&page_size=100&search=无&show_expand_item=1&show_sum_acct_type=1' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| page\_count | integer | 当前页返回行数 |
| rows | List <Row> | 数据行 |
| total\_count | integer | 凭证明细总数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| account\_id | string | 科目id |
| account\_name | string | 科目名称（如有辅助资料，则包含辅助资料名称） |
| account\_no | string | 科目编码（如有辅助资料，则包含辅助资料编码） |
| begin\_bal | string | 期初余额 |
| begin\_bal\_for | string | 期初原币余额 |
| begin\_dc | string | 期初余额方向 1借 -1贷 0平 |
| credit | string | 本期贷方发生 |
| credit\_for | string | 本期贷方发生（原币） |
| currency\_name | string | 币别名称 |
| debit | string | 本期借方发生 |
| debit\_for | string | 本期借方发生（原币） |
| detail\_id | string | 辅助核算组合id >=0包含辅助核算 默认-1全部 |
| end\_bal | string | 期末余额 |
| end\_bal\_for | string | 期末余额（原币） |
| end\_dc | string | 期末余额方向 1借 -1贷 0平 |
| ytd\_credit | string | 本年贷方发生 |
| ytd\_credit\_for | string | 本年贷方发生（原币） |
| ytd\_debit | string | 本年借方发生 |
| ytd\_debit\_for | string | 本年借方发生（原币） |
