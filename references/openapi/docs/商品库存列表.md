# 商品库存列表

### 基本信息

-   用途说明：商品库存列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/inventory

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| aux\_prop\_ids | array | false | 商品辅助属性ID集合 |
| aux\_prop\_numbers | array | false | 商品辅助属性number集合 |
| include\_batch\_kf\_period | string | false | 是否包含批次、保质期；true-是，false-否 |
| include\_package | string | false | 启用整件散包合计,商品ID集合只能有一个 |
| material\_ids | array | false | 商品ID集合，需要返回整件散包需要传入 |
| material\_numbers | array | false | 商品number集合，需要返回整件散包需要传入 |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| page | integer | false | 当前页，默认1 |
| page\_size | integer | false | 每页显示条数默认10 |
| point\_date | string | false | 是否按指定日期查库存，传入正确的日期即为按指定日期查，如：2020-08-20 |
| sp\_ids | array | false | 仓位ID集合 |
| sp\_numbers | array | false | 仓位number集合 |
| stock\_ids | array | false | 仓库ID集合 |
| stock\_numbers | array | false | 仓库number集合 |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/scm/inventory?aux_prop_ids=[]&aux_prop_numbers=[]&include_batch_kf_period=false&include_package=false&material_ids=[]&material_numbers=[]&modify_end_time=1234567890123&modify_start_time=1234567890123&page=1&page_size=100&point_date=2020-08-20&sp_ids=[]&sp_numbers=[]&stock_ids=[]&stock_numbers=[]' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表 |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| aux\_prop\_id | string | 辅助属性ID |
| aux\_prop\_name | string | 辅助属性名称 |
| aux\_prop\_number | string | 辅助属性编码 |
| aux\_qty | number | 辅助基本单位库存数量（只有浮动多单位商品才会返回） |
| batch\_no | string | 批次号 |
| kf\_date | string | 生产日期 |
| kf\_period | integer | 保质期 |
| kf\_type | string | 保质期单位（1：天，2：月，3：年） |
| material\_id | string | 商品id |
| material\_model | string | 商品规格 |
| material\_name | string | 商品名称 |
| material\_number | string | 商品编码 |
| qty | string | 即时库存数量（基本单位） |
| qty\_package | string | 即时库存整件散包合计 |
| sp\_id | string | 仓位id |
| sp\_name | string | 仓位名称 |
| sp\_number | string | 仓位编码 |
| stock\_id | string | 仓库id |
| stock\_id\_is\_allow\_freight | boolean | 仓库启用仓位管理 |
| stock\_name | string | 仓库名称 |
| stock\_number | string | 仓库编码 |
| valid\_aux\_qty | string | 辅助基本单位库存可用量（只有浮动多单位商品才会返回） |
| valid\_date | string | 到期日 |
| valid\_qty | string | 即时库存可用量 |
| valid\_qty\_package | string | 即时库存可用量整件散包合计 |
