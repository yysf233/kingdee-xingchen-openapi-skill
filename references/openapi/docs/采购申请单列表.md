# 采购申请单列表

### 基本信息

-   用途说明：采购申请单列表
-   请求方式：GET
-   请求地址：https://api.kingdee.com/jdy/v2/scm/pur\_request

### 请求参数

#### headers参数

| 参数名称 | 参数示例 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| Content-Type | application/json | true | 固定传：application/json |
| X-Api-ClientID | 205022 | true | 应用id，[创建指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=316e1f5bd9d711ed8e36c17691e84ff5&id=a540e3dcd9d811ed8e3677dc79b66e86&noside=true) |
| X-Api-Auth-Version | 2.0 | true | 固定传:2.0 |
| X-Api-TimeStamp | 1655775240000 | true | 当前时间毫秒时间戳，有效期为5min |
| X-Api-SignHeaders | X-Api-TimeStamp,X-Api-Nonce | true | 固定传：X-Api-TimeStamp,X-Api-Nonce |
| X-Api-Nonce | 1655775240000 | true | 随机正整数 |
| X-Api-Signature | xxx | true | [参考加密规则](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=b5c26557da9d11ed8e36f9799578d2e1&noside=true) |
| app-token | xxx | true | 产品账套级别的token，[获取指引](https://open.jdy.com/#/files/api/detail?index=3&categrayId=5403e0fd6a5811eda819b759130d6d33&id=9076d1a6da9d11ed8e36874bed64e0be&noside=true) |
| X-GW-Router-Addr | https://tf.jdy.com | true | IDC域名，获取appKey返回消息体内的domain字段 |

#### params参数

| 参数名称 | 参数类型 | 是否必填 | 参数说明 |
| --- | --- | --- | --- |
| bill\_close\_state | string | false | 单据关闭状态（所有：“”，未关闭：“C”，已关闭：“S”,手动关闭: H） |
| bill\_no | string | false | 单据编码 |
| bill\_status | string | false | 单据状态（所有：“”，已审核：“C”，未审核：“Z”） |
| create\_end\_time | string | false | 创建时间-结束时间的时间戳(毫秒) |
| create\_start\_time | string | false | 创建时间-开始时间的时间戳(毫秒) |
| dept\_id | array | false | 调入部门 |
| dept\_number | string | false | 部门编码 |
| emp\_id | array | false | 职员ID |
| end\_bill\_date | string | false | 结束日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| material\_model | string | false | 商品规格型号 |
| material\_name | string | false | 商品名称 |
| material\_number | string | false | 商品编码 |
| modify\_end\_time | string | false | 修改时间-结束时间的时间戳(毫秒) |
| modify\_start\_time | string | false | 修改时间-开始时间的时间戳(毫秒) |
| order\_by | string | false | 排序；编码升序：number asc（默认），编码降序：number desc，名称升序：name asc，名称降序：name desc |
| page | string | false | 当前页，默认1 |
| page\_size | string | false | 每页显示条数默认10 |
| pur\_chase\_status | string | false | 采购状态（所有：“”，全部采购：“A”，部分采购：“P”，未采购：“N”，（新增一个采购状态 B：待采购，它等于未采购+部分采购）） |
| show\_material\_total | string | false | 是否统计商品种类和申请数量、批准数量、单据价税合计到单据，默认false |
| start\_bill\_date | string | false | 开始日期（格式：“yyyy-MM-dd”，为空表示不过滤），单据日期 |
| total\_fields | string | false | 自定义合计字段，多个字段用英文逗号隔开 |

#### 请求示例

```bash
curl --location --request  get 'https://api.kingdee.com/jdy/v2/scm/pur_request?bill_close_state=C&bill_no=001&bill_status=C&create_end_time=1234567890123&create_start_time=1234567890123&dept_id=[]&dept_number=001&emp_id=[]&end_bill_date=2022-05-02&material_model=001&material_name=商品名称&material_number=001&modify_end_time=1234567890123&modify_start_time=1234567890123&order_by=createtime desc&page=1&page_size=100&pur_chase_status=A&show_material_total=false&start_bill_date=2022-05-02&total_fields=allamount' \
--header 'Content-Type: application/json' \
--header 'X-Api-ClientID: 205022' \
--header 'X-Api-Auth-Version: 2.0' \
--header 'X-Api-TimeStamp: 1655775240000' \
--header 'X-Api-Nonce: 1655775240000' \
--header 'X-Api-SignHeaders: X-Api-TimeStamp,X-Api-Nonce' \
--header 'X-Api-Signature: xxx' \
--header 'app-token: xxx' \
--header 'X-GW-Router-Addr: https://tf.jdy.com'
```

### 响应参数

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| errcode | integer | 返回码，成功时为0 |
| description | string | 返回信息，成功时为success，失败时为具体信息 |
| data | Data | 业务返回具体对象 |

**Data**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| count | integer | 总计条目 |
| current\_page\_size | integer | 当前页大小 |
| headers | Header | 返回数据头信息 |
| page | integer | 当前页 |
| page\_size | integer | 页大小 |
| rows | List <Row> | 返回数据列表c=\[\] |
| total\_page | integer | 总页数 |

**Row**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| bill\_date | string | 单据日期，yyyy-MM-dd |
| bill\_no | string | 单据编码 |
| bill\_status | string | 单据状态，已审核：“C”，未审核：“Z” |
| create\_time | string | 创建时间 |
| id | string | 单据ID |
| pur\_chase\_status | string | 采购状态（所有：“”，全部采购：“A”，部分采购：“P”，未采购：“N”） |
| whole\_supplier\_id | string | 建议供应商id |
| whole\_supplier\_name | string | 建议供应商名称 |
| whole\_supplier\_number | string | 建议供应商编码 |

**Header**

| 参数名称 | 参数类型 | 参数说明 |
| --- | --- | --- |
| all\_amount | string | 列表价税合计汇总，符合条件所有单据价税合计汇总，包含本页之外的数据 |
